# Excel--After-Sales-Revenue-Rework-Analysis
Evaluated ₹102Cr service operations revealing 40% revenue leakage from rework costs and 73% poor CSAT linked to high TAT; proposed warranty and process optimization actions.

# Business Problem
An EV charging network wanted to evaluate operational performance and understand how downtime, SLA breaches, and discounting were affecting profitability.
Dataset Overview
92 days of charging session data
Each row represents one charging session
Key fields: Region, City, Station Type, Charger Type, Customer Type, Revenue, Profit, SLA Flag

# KPIs Designed
Total Revenue
Total Profit
Profit Margin %
% SLA Breached
Revenue at Risk

# Analysis Approach
Segmented sessions by region and charger type
Analyzed SLA breach patterns by city
Evaluated downtime buckets and discount % impact on profit
Used slicers for interactive analysis

# Key Insights
14% revenue at risk due to failed or cancelled sessions
Delhi and Mumbai showed highest SLA breaches
Certain charger types had higher downtime
Increasing discount % reduced overall profit

# Recommendations
Improve maintenance for high-downtime charger types
Monitor high-breach cities closely
Re-evaluate high discount strategy
